Apple iPhone X 256GB 採用全螢幕設計

Super Retina 螢幕

Apple iPhone X 256GB 機身正反面配備全玻璃，強化層加厚 50%，並歷經七層上色工序，精準控制色調與透明度，還加入光學反射層以強化色彩；其搭載 5.8 吋 Super Retina 螢幕，無論顯示內容是使用 P3 或 sRGB 色彩模式，手機都會自動以該格式顯示。Apple iPhone X 256GB 採用 OLED 面板，讓 HDR 顯示器具備 1,000,000：1 的對比度、高解析度與亮度，並支援廣色域，用戶可觀賞 Dolby Vision 及 HDR10 格式的電影及節目，HDR 照片看起來也更為驚豔，還能瀏覽 iTunes 及 Netflix 等的 HDR 內容。此外，透過 True Tone 顯示技術，可根據你周遭的光線自動調整白平衡，在任何環境都能擁有更好的視覺體驗。


A11 Bionic 處理器

Apple iPhone X 256GB 運行 iOS 11 作業系統，內建 64 位元 Apple A11 Bionic 六核心處理器、256GB ROM，速度提升高達 70%，即使玩 AR 遊戲也保有流暢速度；搭配第二代性能控制器，懂得在性能核心與節能核心之間分配不同任務，能在你需要時提供更多動力，同時給予更強悍的續航體驗。Apple iPhone X 256GB 支援無線充電系統，將手機放置在 AirPower 充電板上任何位置即可充電，用戶不必再受到充電連接線與耳機線的牽絆。


Face ID 臉部解鎖

Apple iPhone X 256GB 搭載 1,200 萬畫素廣角 + 長焦 iSight 攝錄鏡頭，具備光學影像穩定雙鏡頭，低光環境下也能拍攝出色的照片及影片，搭配全新的感光元件、四合一 LED True Tone 閃光燈，提升進光量的同時讓顏色表現更加真實；還內建 Live Photo 效果與全新色彩濾鏡。Apple iPhone X 256GB 搭載 700 萬畫素 TrueDepth 前鏡頭，整合泛光照射器、紅外線鏡頭、測繪點投射器等科技，並具備 Face ID 臉部解鎖技術；Face ID 具有感知功能，只有在你睜開雙眼直視裝置時才會為你解鎖，可防止有人利用照片或面具來蒙騙過關，無論是解鎖、顯示各項通知及訊息、閱讀時保持螢幕明亮、調低鬧鐘 / 鈴聲的音量，或是行動支付，都變得快速、簡單又輕鬆自然。此外，開啟「人像模式自拍」，前景清晰、背景朦朧，隨手拍出賞心悅目的自拍照。
